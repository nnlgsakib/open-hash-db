Hello, OpenHashDB!

This is a test file to demonstrate the content-addressable storage capabilities of OpenHashDB.

The file contains some sample text that will be hashed using SHA-256 and stored in the distributed database.

Features being tested:
- Content addressing with SHA-256 hashes
- File upload and storage
- Content retrieval
- Metadata handling

OpenHashDB provides a modern, developer-friendly alternative to IPFS with:
- Blazing-fast performance
- CLI and REST API interfaces
- Distributed peer-to-peer networking
- Automatic chunking and deduplication
- Content verification and integrity checks
